# LumiQ Lab 第 0 篇文章配图 Prompt 清单

## 统一风格说明

所有配图使用 **扁平插画风格 (Flat Illustration)**，干净线条，现代极简设计。

**配色统一：**
- 背景：深海军蓝 / 暗色调
- 强调色：暖金色 / 琥珀色（代表"光"）
- 辅助色：珊瑚橙、柔和白
- 避免：高饱和荧光色、渐变过多

**通用要求：**
- 无文字、无 logo、无水印
- 画面干净，留白充足
- 适合手机端阅读（公众号正文宽度有限，主体元素不要过细过密）

## 图片 1：封面图

**文件名：** `01-cover.png`
**用途：** 公众号文章封面（会被裁切成正方形小图在列表中显示）
**建议尺寸：** 900 × 383 px（2.35:1）或 900 × 500 px（16:9）
**位置：** 文章顶部

**Prompt（英文，复制到 Midjourney / 通义万象 / 即梦）：**

```
A young woman sitting at a desk in front of a laptop, the screen glowing with warm golden light illuminating her face. The background is dark and muted, creating a strong contrast between the darkness and the light from the screen. Soft particles of light float in the air around her. The scene conveys a sense of discovery and hope. Flat illustration style, clean lines, minimal modern design, warm color palette with deep navy blue background and golden amber highlights. No text, no logos.
```

**Prompt（中文，复制到通义万象 / 即梦 / 可灵）：**

```
一个年轻女生坐在书桌前面对着笔记本电脑，屏幕发出温暖的金色光芒照亮她的脸庞。背景是深色的，与屏幕的光形成强烈对比。空气中漂浮着柔和的光点粒子。画面传达出一种发现和希望的感觉。扁平插画风格，干净的线条，现代极简设计，配色以深海军蓝背景和金色琥珀色高光为主。无文字，无logo。
```

## 图片 2：沉睡的项目

**文件名：** `02-dusty-laptop.png`
**用途：** 第一部分结尾，表现项目被搁置的失落感
**建议尺寸：** 900 × 500 px（16:9）
**位置：** "一、2025年8月..." 部分结尾

**Prompt（英文）：**

```
A laptop sitting on a desk covered with a thin layer of dust, the screen showing a GitHub commit history page with the last commit dated "8 months ago". Next to the laptop is a yellow sticky note with handwritten text. The scene feels abandoned and melancholic, with muted cool blue-gray tones. Dust particles float in the dim light. Flat illustration style, clean lines, minimal modern design, cool desaturated palette. No text on screen, no logos.
```

**Prompt（中文）：**

```
一台笔记本电脑放在书桌上，上面覆盖着薄薄一层灰尘，屏幕上显示着 GitHub 提交记录页面，最后一次提交显示"8个月前"。电脑旁边有一张黄色便利贴，上面有手写文字。整个场景给人一种被遗弃和忧郁的感觉，色调偏冷的蓝灰色。灰尘颗粒在昏暗的光线下飘浮。扁平插画风格，干净的线条，现代极简设计，冷色调低饱和度配色。屏幕上无文字，无logo。
```

## 推荐生成平台

| 平台 | 特点 | 适合哪张 |
|------|------|---------|
| **通义万象 (DashScope)** | 中文理解好，支持自定义尺寸，性价比高 | 全部 |
| **即梦 (Jimeng)** | 字节出品，插画风格出色 | 封面、AI 助手 |
| **Midjourney** | 艺术感最强，但需要订阅 | 封面、结尾 |
| **可灵 (Kling)** | 快手出品，画面干净 | 全部 |
| **智谱 GLM-Image** | 性价比高，支持中文 prompt | 全部 |
