## 卡片 2：故事起点

**文案内容**：

```
那天晚上，我哥哥给我说，现在AI写代码很容易，没学过也可以写代码了。所以我打开 VS Code，决定给自己做一个备忘录的网页。

不是那种复杂的 APP，就是一个最简单的网页：输入文字，点保存，刷新页面还能看见。我用的是 Python 的 Flask 框架——对，就是网上教程最多的那个。

我写了大概 200 行代码。有增删改查，有一个简陋的界面，能跑。

然后我做了一件很多初学者都会做的事：**我把它 push 到 GitHub 上，告诉自己"以后再慢慢加功能"，然后关了电脑。**

想解决但不敢碰的问题：
• 给备忘录打标签，这样我可以分类
• 加个搜索框，不然备忘录多了根本找不到
• 做个好看一点的主题，现在的界面太丑了
• 加个分页，不然加载会越来越慢
• 还有……备份功能？万一数据丢了怎么办
```
