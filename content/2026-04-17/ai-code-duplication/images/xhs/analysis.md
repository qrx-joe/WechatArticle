## 关键要点
1. 三个月前同时开了两个 GitHub 仓库，分别用 Codex 和 Claude 开发
2. Codex 版：代码乱、有 CI/CD、能跑
3. Claude 版：代码优雅、文档完整、但 CI 和回滚没了
4. 功能重叠 > 90%，花三个月得到两个 draft
5. 核心洞察：AI 是镜子，反应的是你的真实需求，不是 AI 的能力
